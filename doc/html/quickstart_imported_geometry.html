Quickstart Guide: Imported Geometry

The purpose of this guide is purely for how to import SWC geometries. It will not cover how to construct networks, or how to interface with networks. For the guide on this please see the previous section on "Custom Geometry"

SWC Files

ELFENN is able to import SWC files to re-create realistic geometries. For our example we will use the stellate-like insterneuron from neuromorpho.org (ID NMO_66468).
We've saved the file in "P7i_WT3.CNG.swc"
warning ('off','all');
cell = ELFENN.Geometry.load_swc("P7i_WT3.CNG.swc");
warning('on', 'all');
If we did not remove warnings we would see a large number of warnings. This is because at a time small sections led to numerical instabilities, this is no longer a problem but the warnings remain as it could be a future issue.
Like the custom geometry, we can plot the cell
plotcell(cell, 'style', 'line')
To see more information about plotting please see the plots guide.

Lowering Resolution

SWC files often have a very high spatial resolution which adds computational complexity which is often unwanted. ELFENN includes a way of decreasing the spatial resolution while maintaining the location of branches and endpoints.
By specifying optionally specifying the spatial resolution to the load_swc method the created cell will be a good approximation to the lower resolution
warning ('off','all');
cell_full = ELFENN.Geometry.load_swc('P7i_WT3.CNG.swc');
cell_low = ELFENN.Geometry.load_swc('P7i_WT3.CNG.swc', 100);
warning ('on','all');
Comparing them shows the spatial resolution has been given to approximately 100um. The red cell is the full reconstructed cell and the blue cell overlayed is the approximation.
plotcell(cell_full, 'style', 'line', 'color', 'red')
plotcell(cell_low, 'style', 'line', 'color', 'blue')
