Quickstart Guide: Connectivity

Cells in ELFENN can be connected through either electrical synapses (gap junctions) or chemical synapses.

Example Network

Like before we'll create an example network to be used throughout this guide.
axon = ELFENN.Section('axon', 'nSeg',5);
cell = ELFENN.Cell('cell1');
cell.connectsection(axon);
Warning: Cell will be a single section
network = ELFENN.Network();
network.addcell(cell, [0,0,0]);
cell.name = 'cell2';
network.addcell(cell, [0,0,100]);
network.complete();

Electrical Synapses

Electrical synapses are the simplest form of connectivity in ELFENN, all thats needed is two compartments and a conductance (mS). This creates a symmetric gap junction between the two cells.
g = 1e-5; % 10nS or 1e-5 mS
network.addelectricalsynapse(network.getcellbyname('cell1').tip, ...
network.getcellbyname('cell2').tip, g);

Chemical Synapses

Chemical synapses (also refered to as autonomous synaoses) can be added just as easily. Unlike electrical synapses these are uni-directional with the syntax that the pre-synaptic index projects to the post-synaptic index.
p = ELFENN.Mechanisms.Synapse.Alpha.default_parameters;
p.gSyn = 2;
p.t0 = [5, 50];
index_pre = network.getcellbyname('cell1').tip;
index_post = network.getcellbyname('cell2').tip;
network.addchemicalsynapse(index_post, index_pre,...
@ELFENN.Mechanisms.Synapse.Alpha.s, p, 'unscaled');
