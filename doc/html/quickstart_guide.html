ELFENN Quickstart Guide

Welcome to the ELFENN Quickstart Guide. This Guide covers how to use the basic features of ELFENN as well as hint at some of the more advanced features. For full examples please see the "Examples" document. The guide is broken up into several section sections to cover the basics of ELFENN. We reccomend doing each guide in sequence.

Geometry

The geometry guide will cover how to build neuron geometries and combine them into a network

Dynamics

The dynamics guide will cover how to convert your standard dynamics to encorporate ephaptic coupling.

Stimulus

This guide will cover basic mechanisms like injected current or extracellular stimuli.

Connectivity

This guide will cover how cells in a network can be connected

Solver

The solver guide will cover how to configure and run the solver

Visualization

The plotting guide will cover how to visualize your simulations
