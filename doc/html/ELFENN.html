ELFENN: A toolbox for modeling ephaptic coupling

Welcome to ELFENN: a MATLAB toolbox for modeling ephaptic coupling. ELFENN stands for ELectric Field Effects in Neural Networks and provides an easy to use interface to the Holt and Koch method of modeling ephaptic coupling (1999 J. Comput. Neurosci.).
If you have not done so already you should run the test suite to make sure everything is working as it should be.
testELFENN()
ELFENN ships with solutions from NEURON and LFPy to check accuracy. If you do not wish to recompute these solutions answer "n" to the prompt. If you do wish to recompute these solutions (not recomended) you must configure Tests/config.json (see Tests/config.example.json for format).
ELFENN has been tested to work with MATLAB 2016b and 2017b, 2018a/b. if an issue does arise. Please contact
Aaron Shifman
ashifman@uottawa.ca
Or raise an issue on Github
ELFENN simulations are created and solved in the following steps
  1. Model geometries are speciefied
  2. Networks are composed out of neurons
  3. Dynamics are added to the network
  4. Simulation is computed
  5. Results are analysed
