Quickstart Guide: Stimulus

For most dynamics, cells are quiescent without some sort of stimulus. There are two types of stimuli in ELFENN, intracellular and extracellular dynamics. ELFENN includes functionality for DC or sinusoidal stimuli. But like dynamics custom plugins are trivial. Furthermore we classify non-autonomous synapses as stimuli - synapses with pre-prescribed spike times.
We will begin with using sinusoidal stimuli from ELFENN and afterwards we'll go over the format for custom functions.

Intracellular Stimulus

We are going to inject a sinusoidal current into the tip of the cell. With a frequency of 1Hz, and amplitude of 100 pA (0.1 nA)
p.amp = 0.1;
p.freq = 1;
p.phase = 0;
handle = @ELFENN.Mechanisms.Stimulus.sinusoidal_stimulus
handle = function_handle with value:
@ELFENN.Mechanisms.Stimulus.sinusoidal_stimulus
network.addintracellularelectrode(network.getcellbyname('cell').tip,...
handle, p, 'unscaled');
The 'unscaled' referes to the fact that total current is defined in nanoamperes. If 'scaled' had been specified the unit would have been uA/cm^2.

Extracellular Stimulus

Point Electrodes

Point electrodes are added much in the same way intracellular electrodes are added except there is no option for scaling, and the location is the [x,y,z] location of the field. Here we add an extracellular point electrode to the 100um above the origin in the z-axis. The parameters 'p' are the same as the previous example and the units for the amplitude are nA
network.addextracellularelectrode([0,0,100],...
@ELFENN.Mechanisms.Stimulus.sinusoidal_stimulus, p);

Plate Electrodes

In the event a uniform field is to be added to the extracellular space ELFENN can compute the field from infinite parallel plates. Unlike point electrodes, the "amplitude" of the field is defined by the field strength in V/m (mV/mm), and the position is defined as the normal vector of the "positive" plate and a point on the plane - along with the separation of the plates.
Here we create a plate set-up with a constant 1mV/mm field with between two electrodes parallel to the ZY-plane at x=50 and x=-50 (100 um apart)
p.amp = 1;
network.addextracellularplate([50, 0, 0], [-1, 0, 0], 100,...
@ELFENN.Mechanisms.Stimulus.dc_stimulus, p);

Non-autonomous synapses

Non-autonomous synapses, or synapses with preset spike times, i.e. a Poisson spike train can be easily added to compartments. Here we add an alpha synapse which triggers at 5ms and 50ms. Again the unscaled flag referes to the fact the conductance is in uS instead of mS/cm^2.
ELFENN includes 3 types of synapses, Alpha functions, exponential synapses, and bi-exponential (exp2) synapses.
synapse_params = ELFENN.Mechanisms.Synapse.Alpha.default_parameters;
synapse_params.gSyn = 1;
synapse_params.t0 = [5, 50];
network.addexternalsynapse(network.getcellbyname('cell').tip, ...
@ELFENN.Mechanisms.Synapse.Alpha.s, synapse_params, 'unscaled');

Creating Custom Stimuli

Stimuli are functions which take 3 inputs: time, state, and a struct of parameters. We show an example here
type("ELFENN/Mechanisms/Stimulus/sinusoidal_stimulus")
function stimCurrent = sinusoidal_stimulus(t, ~, p)
stimCurrent = p.amp * sin(2*pi*p.frequency*t+p.phi);
end
We can see it takes 3 inputs
Time: the current time
y: the state of the system without extracellular potentials in the same order as the ODE state. See "Dynamics" guide for explanation.
p: a struct of parameters for the stimulus such as amplitude
